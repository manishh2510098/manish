# Simple Python Calculator  
![Calculator Preview](https://via.placeholder.com/400x500.png?text=Python+Calculator)  
*A clean, beautiful, and easy-to-understand calculator built with Python and Tkinter.*

---

### Features  
- Basic operations: **+ − × ÷**  
- Additional functions: **% (percentage)**, **± (toggle sign)**, **.** (decimal)  
- Clear button (**C**) to reset  
- Error handling (shows "Error" for invalid input)  
- Responsive and modern GUI  
- 100% pure Python – no external packages required

### Requirements  
- Python 3.6 or higher  
- Tkinter (comes built-in with standard Python)

---

### How to Run  
1. Save the code as `calculator.py`  
2. Open terminal/command prompt  
3. Run the command:  
```bash
python calculator.py
```

That's it! The calculator window will pop up.
